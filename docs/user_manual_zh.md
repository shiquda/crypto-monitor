## 1. 快速入门

### 安装与启动
- **下载**: 请从 [GitHub Releases](https://github.com/shiquda/crypto-monitor/releases) 页面下载最新版本的 `.exe` 安装包。
- **启动**: 双击运行即可。无需管理员权限，默认安装在当前目录或 AppData 中。
- **更新**: 软件内置更新检查功能，点击 `设置 > 关于 > 检查更新` 即可获取最新版本提示。

> **注意**: 首次运行时，Windows SmartScreen 可能会提示风险，这是由于开源软件未购买签名证书所致，点击“仍要运行”即可。

### 界面概览
主界面由三个核心区域组成：
1. **顶部工具栏**: 包含设置、添加交易对、置顶窗口、最小化和关闭按钮。
2. **行情卡片区**: 核心显示区域，每个卡片代表一个交易对。
3. **底部导航栏**: 当关注的币种较多时，使用圆点进行翻页（支持滚轮翻页）。

## 2. 核心功能

### 实时监控
- **数据源**: 支持 **Binance** 和 **OKX** 两大主流交易所。
- **添加交易对**: 点击顶部的 `+` 号，输入如 `BTC-USDT` 即可添加。

<p align="center">
  <img src="./imgs/add-pair.png" alt="添加交易对" width="60%">
</p>

- **查看详情**: 
    - **悬停**: 鼠标停留在卡片上，会浮现该币种的 **24小时最高/最低价**、**成交量** 以及 **迷你K线图**。
    - **双击**: 直接在浏览器中打开该交易对的交易所K线页面。

  <img src="./imgs/hover.png" alt="悬停详情" width="60%">

### 智能预警系统
这是本软件最强大的功能，助您不错过任何行情。
**右键点击**任意卡片，选择 `添加预警 (Add Alert)` 即可配置：

<p align="center">
  <img src="./imgs/add-alert.png" alt="添加预警" width="60%">
</p>

#### 预警类型详解
| 类型 | 说明 | 适用场景 |
| :--- | :--- | :--- |
| **高于 (Above)** | 价格涨破设定值时触发 | 止盈提醒 |
| **低于 (Below)** | 价格跌破设定值时触发 | 止损/抄底提醒 |
| **触及 (Touch)** | 价格**碰到**设定值（±0.1%误差）时触发 | 关键点位关注 |
| **价格阶梯 (Price Step)** | 价格每变动一定数值（如每涨跌 $1000）触发一次 | 追踪大波动行情 |
| **涨跌幅阶梯 (Change Step)** | 24小时涨跌幅每变动一定比例（如每变动 1%）触发一次 | 监控剧烈波动 |

#### 频率设置
- **一次 (Once)**: 触发后自动关闭。
- **重复 (Repeating)**: 满足条件即触发，带有 60秒 冷却时间，防止消息轰炸。

> **提示**: 预警消息通过 Windows 系统原生通知推送，点击通知可直接打开交易页面。

## 3. 个性化体验

### 极简摸鱼模式 (Minimalist View)
专为办公室环境设计，开启后窗口仅保留价格信息，极大降低存在感。

- **开启方法**: `设置 > 外观 > 开启“极简模式”`。
- **交互逻辑**: 
    - 当鼠标**离开**窗口区域时：自动隐藏顶部工具栏和底部翻页器，窗口高度自动收缩，仅显示紧凑的价格卡片。
    - 当鼠标**移入**窗口区域时：所有控件自动浮现，方便操作。

### 视觉与交互习惯
- **涨跌色模式**: 默认为“绿涨红跌”（标准/西式）。如果您习惯国内股市，可在 `设置 > 外观` 中切换为“红涨绿跌”。
- **动态背景**: 开启后，软件背景透明度会随**价格波动剧烈程度**微调，让行情“呼吸”起来。
- **自动轮播**: 开启后，卡片列表将自动按设定的时间间隔（5秒 - 300秒）翻页，适合挂机监控多页币种。

### 置顶与布局
- 点击工具栏的 **图钉 (Pin)** 图标，可使窗口始终保持在最顶层，不被其他工作窗口遮挡。
- 窗口支持随意拖动位置，并在关闭时自动记忆上次位置。

## 4. 高级设置

点击工具栏的 **齿轮** 图标进入设置中心。

<p align="center">
  <img src="./imgs/gear.png" alt="设置" width="30%">
</p>

### 网络与数据源
- **数据源切换**: 默认为 OKX。如果无法访问，可切换为 Binance。
- **价格计算基准**: 
    - **24h Rolling**: 无论何时查看，均显示过去24小时的变化（默认）。
    - **UTC-0**: 显示从当日 UTC 时间 0点（北京时间早8点）至今的变化，适合日内交易者。
- **代理设置**: 如果直连交易所API受阻，请在 `设置 > 网络` 中配置 HTTP 或 SOCKS5 代理。

### 悬停卡片配置
您可以在 `设置 > 外观` 中精细控制鼠标悬停时的显示内容：
- **显示内容**: 可独立开启/关闭“迷你图表”或“详细统计数据”。
- **图表周期**: 支持 `1h`, `4h`, `12h`, `24h`, `7d` 五种周期。
- **图表缓存**: 在高级选项中可调整图表缓存时间（10s - 10min），调大该值可节省流量并减少API请求。

### 备份与迁移
- **导出配置**: 在设置首页底部点击 `导出配置`，可将您保存的所有交易对和复杂的预警设置备份为 JSON 文件。
- **导入配置**: 重装系统或更换电脑后，可一键导入恢复。

## 5. 常见问题 (FAQ)

**Q: 为什么点击“检查更新”后没有反应？**  
A: 更新检查连接的是 GitHub API。如果您的网络无法访问 GitHub，可能导致检失败。如有更新，它会弹窗提示并引导您去浏览器下载，**不会**后台静默安装。

**Q: 为什么收不到预警通知？**  
A: 请检查 Windows 系统的“专注助手”或“通知设置”，确保未屏蔽 Crypto Monitor 的通知权限。

**Q: 软件有系统托盘图标吗？**  
A: 目前版本暂未实现系统托盘功能，最小化后会停留在任务栏。

**Q: 如何彻底重置软件？**  
A: 在设置界面底部点击 `恢复默认设置`，或手动删除 `%APPDATA%/crypto-monitor` 文件夹。
