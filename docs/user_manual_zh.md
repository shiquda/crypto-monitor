# Crypto Monitor 用户手册

欢迎使用 Crypto Monitor，这是一款专为桌面设计的现代优雅的加密货币价格监控器。

## 1. 简介
Crypto Monitor 允许您直接从桌面追踪来自 OKX 和 Binance 等主流交易所的实时加密货币价格。它拥有令人惊叹的 Fluent Design 界面，与 Windows 11 完美融合。

### 主要功能
- **实时监控**：通过 WebSocket 进行实时价格更新。
- **流畅设计**：具有亚克力效果的精美 UI，支持浅色和深色主题。
- **智能预警**：设置价格阈值和阶梯预警，随时掌握行情。
- **桌面集成**：支持系统托盘、原生通知和置顶模式。
- **可定制**：极简界面，支持调整大小和拖动窗口。

![主界面](./imgs/crypto-monitor.png)

## 2. 安装与设置

### 前提条件
- Windows 10 或 11（推荐）。
- 互联网连接。
- Python 3.10+（如果是从源码运行）。

### 运行应用程序

您可以在 Github Release 页面找到预构建的可执行文件。

如果您有可执行文件 (EXE)：
1. 双击 `Crypto Monitor.exe`。

如果是从源码运行：
1. 确保您已安装 `uv` 或 `pip`。
2. 安装依赖项：`uv sync` 或 `pip install -r requirements.txt`。
3. 运行应用程序：
   ```bash
   uv run main.py
   ```

## 3. 入门指南

### 主界面
主界面紧凑简洁，以卡片形式显示您追踪的加密货币交易对。
- **工具栏 (顶部)**：访问设置、添加交易对、置顶窗口、最小化和关闭。
- **加密货币卡片**：每张卡片显示符号、当前价格和 24 小时涨跌幅。
- **分页 (底部)**：如果您有很多交易对，可以使用底部的圆点进行翻页。

### 添加交易对
1. 点击工具栏中的 **(+) 添加** 按钮。
2. 输入交易对符号（例如 `BTC-USDT` 或 `ETH-USDT`）。
3. 点击 **添加**。新交易对将出现在主卡片列表中。

![添加交易对](./imgs/add-pair.png)

### 移除交易对
1. 将鼠标悬停在您想要移除的卡片上。
2. 点击卡片上出现的 **垃圾桶** 图标。

## 4. 功能使用

### 查看详情
- **悬停**：将鼠标悬停在卡片上可查看详细的 24 小时统计数据（最高价、最低价、成交量）。
- **图表**：悬停卡片中会显示迷你价格趋势图。
- **打开交易所**：双击卡片可在默认浏览器中打开该交易对的交易所网页。

![悬停详情](./imgs/hover.png)

> 注意：您可以在 设置 > 外观 > 悬停卡片 中管理悬停卡片的内容。

### 管理预警
您可以设置预警以接收价格变动通知。
1. **右键点击** 加密货币卡片。
2. 选择 **添加预警 (Add Alert)**。
3. 配置您的预警：
   - **目标价格 (Target Price)**：设置触发预警的具体价格。
   - **条件 (Condition)**：选择 `高于 (Above)`、`低于 (Below)` 或 `触及 (Touch)`。
   - **频率 (Frequency)**：
     - **一次 (Once)**：预警触发一次后禁用。
     - **重复 (Repeating)**：每次满足条件时都触发预警（有冷却时间）。
   - **阶梯预警 (Step Alerts)**：每当价格变动一定金额时通知（例如，每上涨 $100）。

![添加预警](./imgs/add-alert.png)

要管理现有预警，请右键点击卡片并选择 **查看预警 (View Alerts)**。

### 置顶显示
点击工具栏中的 **图钉** 图标，可让 Crypto Monitor 窗口始终保持在其他窗口之上。

## 5. 设置

点击工具栏中的 **齿轮** 图标打开设置。

![设置图标](./imgs/gear.png)

### 常规
- **数据源**：在 **Binance** 和 **OKX** 之间切换。请注意，数据可用性可能因地区而异。
- **语言**：在 英语 和 中文 之间切换。

### 外观
- **主题**：在 **浅色 (Light)** 和 **深色 (Dark)** 模式之间切换。
- **动态背景**：启用/禁用基于价格波动强度的背景透明度变化。

### 网络
- **代理**：如果您处于受限网络中，可以配置代理。支持 HTTP 和 SOCKS5。

### 关于
- 查看版本信息。
- **打开日志目录**：快速访问日志文件以进行调试。
